### Open task manager only
```
Raden Booster.exe --taskmgr
```

### Open cleaner only
```
Raden Booster.exe --cleaner
```

### Boost RAM
```
Raden Booster.exe --boost
```

### Open as tray icon
```
Raden Booster.exe --hide
```
